# Retro Space Defender

A minimal retro arcade space shooter built with HTML5 Canvas and vanilla JavaScript.

## How to Run

1. Open `index.html` in a web browser, or
2. Serve the folder with a local server (e.g. `npx serve .` or `python -m http.server 8000`) and open the URL

## Controls

- **WASD** or **Arrow keys** — Move
- **Space** — Shoot

## Gameplay

- Destroy incoming invaders before they reach the bottom
- Each enemy that reaches the bottom costs one life
- You have 3 lives
- Score 100 points per enemy destroyed
