# RT PM Skills

This folder contains 70 auto-discoverable skills for Claude Code.

## How It Works

Claude automatically scans this folder at session start. When you describe a task that matches a skill's description, Claude loads the full instructions automatically.

**You don't need to remember command names** — just describe what you want:
- "Help me analyze this competitor" → loads competitive-profile-builder
- "Extract jobs-to-be-done from this interview" → loads jtbd-extractor
- "Create a PRD for this feature" → loads prd-generator
