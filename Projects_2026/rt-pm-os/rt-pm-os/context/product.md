# Product Overview

## Programs

### Software Engineering Bootcamp
Full-stack development program covering HTML, CSS, JavaScript, Python, React, SQL, Flask, and cloud tools. Available in full-time, part-time, and self-paced formats.

### Data Analytics Bootcamp
Python-focused data program covering Pandas, NumPy, Matplotlib, scikit-learn, and introductory machine learning. Deeper Python instruction than most competing data bootcamps. ~4 months duration.

### Cybersecurity Bootcamp
Self-paced program covering fundamentals, threat analysis, ethical hacking, incident response, risk management, and emerging technologies. Hands-on projects and industry-focused modules.

### Quality Assurance Bootcamp
QA testing fundamentals for aspiring quality assurance professionals.

- **Flexible formats** — Full-time, part-time, self-paced, and online options
